# 🏪 Shop Billing & Management System

A simple and powerful **Shop Billing & Inventory Management App** built using React and LocalStorage.

---

## 🚀 Features

### 🧾 Billing System
- Create bills with multiple products
- Auto bill number generation
- Discount & GST calculation
- Profit calculation
- Print bill support
- Draft auto-save & restore

### 📦 Product Management
- Add / Update / Manage products
- Real-time stock tracking
- Low stock alerts

### 💸 Damage Management
- Record damaged products
- Auto reduce stock
- Track total loss

### 📊 Dashboard
- Today Sales
- Today Profit
- Damage summary
- Net Profit
- Top selling products
- Low stock alerts

### 📈 Sales Report
- Filter by:
  - Today / Weekly / Monthly / Yearly
  - Custom date range
- Search by bill number / customer
- Total Sales / Profit / Damage summary
- Detailed bill-wise table

### 👥 User Management
- Owner / Manager / Worker roles
- Role-based access control
- Login & Registration system

### ⚙️ Settings
- Shop details (Name, Address, Phone, GST)
- Manage Workers & Managers

---

## 🛠️ Tech Stack

- **Frontend:** React
- **UI:** Tailwind CSS
- **Storage:** LocalStorage
- **Routing:** React Router
- **Icons:** React Icons

---

## 💡 Key Highlights

- Fully offline app (no backend required)
- Fast and lightweight
- Clean and simple UI
- Real-time stock updates
- Profit & loss tracking
